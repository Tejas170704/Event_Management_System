# Event Management System

A simple and professional **Event Management System** built using **HTML, CSS, and JavaScript**.  
This project allows users to add, view, and delete events with optional image upload support.  
All event data is stored in the browser using **LocalStorage**.

---

## Features

- Add new events
- Store event name, date, location, and description
- Upload event image
- Preview uploaded image before saving
- Delete events
- Events are automatically sorted by date
- Data is saved in browser LocalStorage
- Toast notifications for actions
- Responsive and modern UI

---

## Technologies Used

- **HTML5** – Structure of the web page
- **CSS3** – Styling and responsive design
- **JavaScript (Vanilla JS)** – Functionality and LocalStorage handling
